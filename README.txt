# Telegram Channel Downloader (Interactive CLI)

## Instructions:
1. Open main.py and edit:
   - api_id, api_hash, phone_number with your actual Telegram developer credentials.

2. Run the script:
   python main.py

3. When prompted, enter:
   - The channel username or invite link
   - The number of messages to download

Media will be saved in /downloads/
Text will be saved in /messages/messages.txt
